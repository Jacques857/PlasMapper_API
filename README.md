# *Endpoints*:
## *Get Features*:
#### Gets promoters, terminators, selectable markers, reporter genes, replication origins, regulatory sequences, affinity tags, and miscellaneous features.
### *Input JSON fields*:
* sequence : String
### *Output JSON fields*:
* features: Array of Feature Objects
#### Feature has properties:
* name : String
* start : Integer
* stop : Integer
* legend : String
